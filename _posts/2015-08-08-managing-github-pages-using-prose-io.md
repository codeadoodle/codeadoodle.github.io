GitHub provides freely hosted public web pages for users, organizations, and repositories. It is powered by Jekyll, a Ruby-based website generator that convert content written in Markdown into a complete static website. However, managing and updating content using Jekyll could be [frustrating and inconvenient](http://www.smashingmagazine.com/2014/08/build-blog-jekyll-github-pages/) as you may need to deal with command line interface to build and publish content into GitHub.

Enter [Prose.io](http://prose.io/) which provides a simple online content authoring environment and management directly for GitHub.
