Managing GitHub Pages using Prose.io

Aug 8, 2015

1 minute read

GitHub provides freely hosted public web pages for users, organizations, and repositories. It is powered by Jekyll, a Ruby-based website generator that convert content written in Markdown into a complete static website. However, managing and updating content using Jekyll could be frustrating and inconvenient as you need to deal with command line interface to build and publish content into GitHub.

Enter Prose.io which provides a simple online content authoring environment and management directly for GitHub. Assuming you have setup or forked your Jekyll templates all you need to do is

  • Authorize Prose.io
  • Add Prose configuration into your site _config.yml

Prose.io is open source and built with Backbone.js and therefore you can run it on your own server. It is not limited for blogging as you get access to all your repositories and practically edit any file you want.
